# NFT Minting dApp

This application is going to be used for minting images into blockchain.

## Technologies

* React.js
* Redux
* Web3.js
* Solidity


## Getting Started

### Installing

* The project can be downloaded by cloning the main branch.
```
npm install
```

### Executing program

```
npm start
```
